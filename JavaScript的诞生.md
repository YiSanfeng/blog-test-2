## JavaScript 的10个设计缺陷
* 不适合开发大型程序
* 非常小的标准库
* null和undefined两者非常容易混淆，但是含义完全不同
* 全局变量难以控制
* 自动插入行尾分号
* 加号运算符
* NaN
* 数组和对象的区分
* == 和 ===
* 基本类型的包装对象
